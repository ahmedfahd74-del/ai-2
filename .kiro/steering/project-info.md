# Project Info

## Purpose
- This repository contains Pine Script v6 trading scripts for TradingView
- Scripts include indicators, strategies, and libraries for technical analysis and automated trading

## Platform
- TradingView (www.tradingview.com)
- Pine Script v6

## File Organization
- Place indicator scripts in `indicators/` directory
- Place strategy scripts in `strategies/` directory
- Place library scripts in `libraries/` directory
- Use descriptive filenames in kebab-case (e.g., `rsi-divergence.pine`, `mean-reversion-strategy.pine`)
- Use `.pine` file extension for all scripts
